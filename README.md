# 🤖 PDF Question Answering System

An AI system that reads any PDF and answers questions from it in plain English.
Built using Python, LangChain, OpenAI API, FAISS, and FastAPI.

## 🧠 Tech Stack
- Python
- LangChain
- OpenAI API
- FAISS (Vector Database)
- FastAPI
- React (Frontend)
